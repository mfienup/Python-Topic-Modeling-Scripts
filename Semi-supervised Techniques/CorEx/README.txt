This directory contains subdirectories:
1) CorEx_Scripts - scripts to analyze chat corpus by performs 
topic modeling utilizing CorEx (Correlation Explanation)

2) Output - output files for topics found using various chat
corpus preprocessing

3) CorEx_Reference_Material - some background articles on 
CorEX (Correlation Explanation)

Here we are used the CorEx package available at GitHub:
https://github.com/gregversteeg/corex_topic
