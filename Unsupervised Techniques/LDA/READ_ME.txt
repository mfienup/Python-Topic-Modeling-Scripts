This directory contains subdirectories each containing 
scripts, data files, and output files for analyzing the 
chat corpus by performs topic modeling utilizing LDA 
(Latent Dirichlet Allocation).

The two subdirectories are:
1) sklearn_LDA - contains the analysis of the chat corpus 
by performs topic modeling utilizing scikit-learn.org (sklearn) 
LDA module.

2) PyMallet_LDA - ontains the analysis of the chat corpus 
by performs topic modeling utilizing the LDA implementation from 
GitHub PyMallet at:
https://github.com/mimno/PyMallet

The LDA code is based on their lda_reference.py code written 
in Python
